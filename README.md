# Cave exterior prototype

Desktop-Chrome first-person cave journey powered by Babylon.js and Havok. The normal URL starts outside the mountain and continues through the main door, carved corridor, and specialization hall. The Linux Kernel & eBPF door opens into one compact portfolio chamber; the other four specialization doors remain closed.

Click to enter Pointer Lock. Use WASD or arrow keys to move, the mouse to look, and Enter only when a prompt is visible. The locked controller uses 4.5 speed, 15 acceleration, 13 deceleration, 1.07 FOV, and .002 mouse sensitivity. Review routes include `/?review=cave-entrance`, `/?review=cave-interior`, `/?review=specialization-hall`, `/?review=linux-ebpf-room`, `/?review=door-interaction`, `/?review=movement&profile=locked`, and `/?review=chamber`.
