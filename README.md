# Cave exterior prototype

Desktop-Chrome first-person cave journey powered by Babylon.js and Havok. The normal URL starts outside the mountain and continues through the main door, carved corridor, and five closed specialization doors.

Click to enter Pointer Lock. Use WASD or arrow keys to move, the mouse to look, and Enter only when the main-door prompt is visible. The locked controller uses 4.5 speed, 15 acceleration, 13 deceleration, 1.07 FOV, and .002 mouse sensitivity. Review routes include `/?review=cave-entrance`, `/?review=cave-interior`, `/?review=specialization-hall`, `/?review=door-interaction`, `/?review=movement&profile=locked`, and `/?review=chamber`.
