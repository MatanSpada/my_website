# Cave prototype conventions

- Keep application modules in `src/game` small and single-purpose.
- Run `npm run build`, `npm test`, and `npm run test:e2e` before delivery.
- Desktop Chrome is the only target for CAVE-001.
- Do not add sound, mobile controls, cave content, doors, analytics, or backend services in CAVE-001.
- Never read or stage `matan_spada_cv.pdf`.
