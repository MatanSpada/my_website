# CAVE-001 movement comparison

| Profile | Character | Result |
| --- | --- | --- |
| grounded | Firm acceleration, readable bob, conservative FOV | Provisional default |
| responsive | Faster acceleration and wider FOV | Useful comparison baseline |
| cinematic | Slower, larger visual motion | Too soft for default |

`grounded` is provisionally selected because it most clearly communicates a physical standing player without visual float. CAVE-002 will lock values after deliberate feel review.
